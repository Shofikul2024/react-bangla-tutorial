### Reducer কি এবং React-এ কীভাবে কাজ করে?

Reducer মূলত একটি ফাংশন যা একটি নির্দিষ্ট নিয়ম অনুসারে **state** পরিবর্তন করে। এটি React-এ state ম্যানেজমেন্টের জন্য ব্যবহৃত হয়, বিশেষ করে যখন state এর উপর অনেক action কাজ করে এবং state ম্যানেজ করা জটিল হয়ে যায়।

### Reducer-এর মৌলিক ধারণা

Reducer ফাংশন নিচের দুইটি জিনিসের উপর কাজ করে:

1. **বর্তমান state** (যে অবস্থায় state এখন আছে)।
2. **action** (কোন পরিবর্তন বা ইভেন্ট যেটা state-কে আপডেট করবে)।

Reducer একটি নতুন state রিটার্ন করে এবং কখনোই পূর্বের state সরাসরি পরিবর্তন (mutate) করে না।

### Reducer কিভাবে কাজ করে: উদাহরণ দিয়ে

1. **Reducer Function এর গঠন**:

   ```javascript
   function reducer(state, action) {
     switch (action.type) {
       case "INCREMENT":
         return { count: state.count + 1 };
       case "DECREMENT":
         return { count: state.count - 1 };
       default:
         return state; // যদি কোনো action match না করে, বর্তমান state ফেরত দেয়
     }
   }
   ```

2. **useReducer Hook ব্যবহার করা**:
   React `useReducer` হুক সরাসরি Reducer ফাংশন ব্যবহার করার সুযোগ দেয়। এর মাধ্যমে আমরা state এবং একটি dispatch ফাংশন পাই যা action গুলোকে reducer ফাংশনে পাঠায়।

   ```javascript
   import React, { useReducer } from "react";

   function reducer(state, action) {
     switch (action.type) {
       case "INCREMENT":
         return { count: state.count + 1 };
       case "DECREMENT":
         return { count: state.count - 1 };
       default:
         return state;
     }
   }

   function Counter() {
     const initialState = { count: 0 }; // Initial state
     const [state, dispatch] = useReducer(reducer, initialState);

     return (
       <div>
         <p>Count: {state.count}</p>
         <button onClick={() => dispatch({ type: "INCREMENT" })}>+</button>
         <button onClick={() => dispatch({ type: "DECREMENT" })}>-</button>
       </div>
     );
   }

   export default Counter;
   ```

### Reducer-এর কাজের ধাপ

1. **State**: `state` হল ডেটার বর্তমান অবস্থা। যেমন `{ count: 0 }`।
2. **Action**: `action` হল একটি object যা বলে দেবে state কীভাবে পরিবর্তন হবে। উদাহরণ: `{ type: 'INCREMENT' }`।
3. **Dispatch**: `dispatch` হলো সেই ফাংশন যার মাধ্যমে আমরা reducer ফাংশনকে জানাই কী action নিতে হবে।
4. **Reducer Logic**: Reducer ফাংশন আগের state এবং action দেখে নতুন state তৈরি করে।

### উদাহরণ:

#### 1. Initial State:

```javascript
const initialState = { count: 0 };
```

#### 2. Action Structure:

```javascript
{
  type: "INCREMENT";
}
```

#### 3. Dispatch Action:

```javascript
dispatch({ type: "INCREMENT" });
```

#### 4. New State:

Reducer ফাংশন পুরানো state এবং action দেখে নতুন state রিটার্ন করবে।

```javascript
{
  count: state.count + 1;
}
```

### কেন Reducer ব্যবহৃত হয়?

1. যখন একটি কম্পোনেন্টে state ম্যানেজমেন্ট জটিল হয়ে যায়।
2. একই state-এ অনেক action পরিচালনা করতে হয়।
3. `useState` এর চেয়ে বেশি নিয়ন্ত্রিত মেকানিজম দরকার।

### সহজ ভাষায় উদাহরণ

একটি ব্যাংকের হিসাবে টাকা জমা এবং তোলা পরিচালনা করার জন্য আমরা reducer ব্যবহার করতে পারি।

```javascript
function reducer(state, action) {
  switch (action.type) {
    case "DEPOSIT":
      return { balance: state.balance + action.amount };
    case "WITHDRAW":
      return { balance: state.balance - action.amount };
    default:
      return state;
  }
}

const initialState = { balance: 1000 };

// Example Dispatches
dispatch({ type: "DEPOSIT", amount: 500 }); // ব্যালেন্স হবে 1500
dispatch({ type: "WITHDRAW", amount: 200 }); // ব্যালেন্স হবে 800
```

### উপসংহার

1. Reducer হলো একটি ফাংশন যা `state` এবং `action` নিয়ে নতুন state তৈরি করে।
2. এটি immutable (পরিবর্তনহীন) state ম্যানেজ করে।
3. `useReducer` হুক ব্যবহার করে এটি React-এ সহজে ব্যবহৃত হয়।
4. এটি এমন পরিস্থিতিতে কার্যকর যেখানে state-এ অনেক action পরিচালনা করতে হয়।

আপনার যদি কোনো জায়গায় বুঝতে সমস্যা হয়, জানাতে পারেন। আরও সহজ করে ব্যাখ্যা করবো।
